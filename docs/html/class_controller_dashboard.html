Detailed Description

Classe ControllerDashboard - Controller pour le dashboard.

Classe qui contient les méthodes pour afficher le dashboard. Contient aussi les méthodes pour la connexion et la déconnexion de l'utilisateur, et pour la création/modification/suppression d'un projet.
